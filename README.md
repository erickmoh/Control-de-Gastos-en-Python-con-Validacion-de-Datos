# Control de Gastos en Python

Este es un programa simple en Python que permite calcular gastos mensuales y dinero restante.

## Funcionalidad
- Solicita ingreso mensual
- Solicita gastos (renta, comida, transporte)
- Calcula gastos totales
- Muestra dinero restante

## Tecnologías
- Python
